# Real-Time Weather Application

A modern, responsive weather dashboard that fetches live data from a third-party weather API. It features dynamic UI updates and switches background imagery based on current temperatures.

**Live Demo:** [View Project Live](https://mostafaosman9439.github.io/Weather-App/)

---

## Features

* **Live Data Fetching:** Retrieves real-time temperature, city information, and weather conditions (e.g., Cairo, Egypt).
* **Glassmorphic UI:** Features a sleek, modern glass-morphic card overlay with enhanced `backdrop-filter: blur()` effects.
* **Error Handling:** Built-in robust error management to catch network issues and missing data.

---

## Tech Stack Used

* **HTML5 & CSS3:** Semantics and responsive layout styling with modern UI trends.
* **JavaScript (ES6):** Advanced Asynchronous JS using `Async/Await`, `Fetch API`, and JSON data parsing.
